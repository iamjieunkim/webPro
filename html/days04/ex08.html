[html 코딩 규칙 (()) 가이드]
        1. 문서 유형 선언문 : 대문자 권장 DOCTYPE.html
        2. 요소(태그)명 : 소문자 권장
        3. 모든 요소가 닫기 태그가 필요한 것은 아니다.
        meta, br, hr, img x
        4. 속성명="속성값"
           속성명='속성값'
           속성명=속성값
           class="active sist18 box"
                =이라는 연산자를 사용할때는 앞, 뒤 모두 공백을 사용하지 말고 =을 주자!
        5. p태그
        p태그는 반드시 닫기 태그를 이어붙이자.
        6.   URL ****반드시 대소문자 구분하자 ****
        7. html 파일 : .html        .htm
        css 파일 :      .css
        javascript:   .js
