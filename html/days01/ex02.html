  1. 인터넷(Internet)
    1. 전 세계를 연결하는 국제 정보 통신망
    2. ARPA(Advanced research projects Agency) 1969년에 웹에 모태가 되는 ARPANET 개발.
    3. 군사용과 민간용으로 구분
    4. 민간용으로 사용되던 네트워크가 현재의 인터넷이 되었다.

  2. 웹(Web)
    1. 인터넷에서 작동되는 서비스
    2. 인터넷에 연결된 컴퓨터를 통해서 사람들이 정보(영상, 문자 등등)을 공유하는 공간
    3. 1991년 등장
    4. 1989년 유럽 공동 원자핵 연구소(CERN)의 "팀버너스 리"가 하이퍼 링크를 포함하는 문서의 개념을 제안 -> 1991년 "월드 와이드 웹(www)"을 개발해서 배포했다.

  3. W3C(World Wide Consortium)
    1. 국제 웹 표준화 단체(기구)
    2. 1994년 10월 설립(팀버너스리 창설)
    3. WWWW의 표준안 제작과 새로운 표준안 제안, 기술의 공유를 통해 WWW의 기술적, 사회적 확장을 위해 구성된 세계적 단체
    4. html5, css3 등등 표준화
    5. 강제성 X

  4. 웹 브라우저
    1. 인터넷에서 정보를 검색(공유)하는 응용 프로그램
    2. 최초 웹 브라우저 - 모자이크 (1993년 ) -> 후에 "넷스케이프" 발표
    3. W3C의 표준을 무시하며 경쟁이 일어나게 됨~~~
    4. MS사 - 인터넷 익스플로러(IE) - PC에 강제로 설치 넷스케이프사 - 넷스케이프웹 브라우저 : :
    5. 웹 브라우저 사이에 발생하는 기술적 전쟁 - 1차 웹 브라우저 전쟁 (1994 ~ 1998년 까지 진행.) -> MS 사의 IE승리

  5. 플러그인
    • 웹의 발전속도를 웹브라우저가 따라가지 못해서 기업이 개발을 하게 됨
    • MS사의 액티브엑스, 어도비사의 플래시(Flash)등등
    • 웹 브라우저와 연동되는 특정 프로그램을 사용자의 PC에 추가로 설치해 웹 브라우저의 기능을 확장하는 방법

  6. 웹2.0 - 서로 다른 사용자가 함께 새로운 컨텐츠를 창조할 수 있는 시대

  7. 웹3.0 - 인터넷에 엄청난 양의 정보 중에 내가 지금 필요한 정보와 지식만을 추출해서 보여주는 맞춤형 웹 시대 (개인화, 지능화, 상황인식 등 웹 3.0의 대표 키워드)

  8. 플러그인 문제 - 액티브 XX와 같은 플러그인이 들어가 있는 웹사이트가 무거워진다. -> 해결 : W3C에서 html 표준제안서가 제출되었는데 (MS사가 거절함.)

  9. WHATWG
    1. Web Hypertext Application Technology Working Group
    2. MS사(IE) 제외 + 웹 브라우저 제공하는 기업 설립
    3. 2004년 6월 html5 표준제정
    4. 새로운 웹 표준인 [Web Application 1.0 표준]을 작성
    5. W3C 또한 [Web Application 1.0 표준] 채택~

  10. 2차 웹 브라우저 전쟁 MS사(IE) < WHATWG h5, c3 X 승리 h5, c3 O

  11. 웹 표준의 변환
    1. W3C Application 1.0 표준(WHATWG)을 html5 표준으로 변경
    2. W3C + WHATWG => HTML W/G을 결성 -> html 5 표준으로 작성
    3. 정식 권고안(2014년)
