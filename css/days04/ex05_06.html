background-origin 속성

    1.background-origin : 배경이미지의 위치를 지정하는 속성이다. 
        -테두리 왼쪽 상단 : border-box
        -패딩 왼쪽 상단(기본값) : padding-box
        -콘텐츠 왼쪽 상단 : content-box
    1.2 background-position : 배경이미지의 위치 지정.

    2.background-clip 속성
        -배경의 페이팅 영역을 지정하는 속성
            border-box(기본값) 배경이 테두리의 바깥쪽 가장자리에 그려진다.
            padding-box 배경이 패딩의 바깥쪽 가장자리에 그려진다.
            content-box 콘텐츠 상자 내에서 배경이 그려진다.
