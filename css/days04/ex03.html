css의 특이성(특수성)

    동일한 요소를 여러개의 css규칙이 적용될 때 운선 순위 결성시 사용되는 특성-> 특이성또는 특수성이라고 한다.
    특이성은 점수로 정해진다.

    1.인라인식 스타일 시트 적용방법 : 1000점
    2.
        *{}    0점
    태그명, ::의사요소:1점
    클래스명, 의사클래스명, 속성선택자 : 10점
    #아이디명                    : 100점

    예)
    #navbar p#demo             : 100 + 1 + 100
    {
        color: red;
    }
    #demo{
        color: blue;           : 100
    }

    3. 예외
    !important 규칙
