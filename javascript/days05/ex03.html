[ js 배열(array)]
  1. 배열 개념 : 둘 이상의 값을 보유할 수 있는 특수한 변수.
  2. 배열 사용 목적
  3. 배열 선언 형식
     자바 :   int [] m = new int[3];
                 m[0] = 1;
                 m[1] = 2;
     배열 초기화    int [] m =    { 1, 2 };         
     js     :   var n = [];
                  n[0] = 1;
                  n[1] = 2;
       배열 초기화  var n  =    [ 1, 2 ]; 
       
    3-2. js 배열 선언 2번째 형식
    var n = new Array();  ==      var n = []; (권장)
    var n = new Array(1,2) ==   var n  =  [ 1, 2 ](권장);
    
    4. 배열 선언할 때 키워드는   let, var, [const ] 권장
